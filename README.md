# Project SPA

1. Download and install Node JS v14.15.1 LTS [link](https://nodejs.org/dist/v14.15.1/node-v14.15.1-x64.msi)
2. Install Angular CLI, command
`npm install -g @angular/cli`
4. In project folder, install dependencies
`npm install`
5. Start project
`ng-serve`
6. Open
http://localhost:4200/
